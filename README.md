# mobil2mealdb
Fejlesztés mobil eszközökre II. beadandó feladat

Egy Androidos alkalmazás megvalósítása a themealdb(https://www.themealdb.com/) API-jávak segítségével, amely a következő funkciókat fogja megvalósítani:
Receptek listázása
Receptek listázása kategóriák, alapanyagok, vagy országok alapján
Receptek részle
tes nézete
Receptek keresése
Kedvencek menü
Saját recept hozzáadása helyileg tárolva
